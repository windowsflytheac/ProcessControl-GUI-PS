# ProcessControl-GUI-PS

**PowerShell GUI to list, select, and terminate Windows processes with logging and warnings.**

---

## ⚠️ WARNING

This tool is **more powerful than the built-in Task Manager**.  
Killing critical processes can crash your device or cause **data loss**.  
Only use this if you **know what you are doing**.  
You are fully responsible for any consequences.

---

## Features

- Lists all running processes with **Name** and **PID**.  
- Select one or more processes to attempt termination.  
- **Kill other processes** option for advanced users.  
- Big warning label displaying your **device name**.  
- Confirmation dialog before killing processes.  
- Success/Failed modal dialog after each operation.  
---

**Running the Program in Administrator Guide**

1. Open Powershell as Administrator
2. Go to your downloads folder
3. Run ./Process-Killer-TaskDialog.ps1
